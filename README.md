# Task 1 - Image Colorization

## 🔧 Model Description
This task performs grayscale to color image colorization using a custom CNN trained on landscape images. It is tested on different categories like animals and portraits.

## 📊 Evaluation Metrics
- Avg PSNR: 24.37 dB
- Avg SSIM: 0.9235
- Avg MAE : 0.047922
- Accuracy: 77.68%
- Precision: 0.9299
- Recall: 0.9118
